Política de Privacidade

SPX CT Cleaner — Extensão Chrome

Última atualização: 05 de março de 2026

1. Informações Gerais

A extensão SPX CT Cleaner é uma ferramenta de produtividade interna destinada a analistas de logística que utilizam a plataforma SPX (Shopee). Sua finalidade é permitir o cancelamento em massa de Calculation Tasks (CTs) pendentes. Esta política descreve como a extensão lida com dados do usuário.

2. Dados Coletados

A extensão NÃO coleta, armazena, transmite ou compartilha nenhum dado pessoal do usuário.

Especificamente, a extensão:

  • Não coleta informações pessoais identificáveis
  • Não utiliza cookies próprios
  • Não possui servidores externos ou backend
  • Não transmite dados para terceiros
  • Não realiza rastreamento ou analytics

3. Dados Processados Localmente

A extensão processa os seguintes dados exclusivamente no navegador do usuário, de forma local e temporária:

  • IDs de Calculation Tasks (CTs): Lidos diretamente da tabela exibida na página do SPX para identificar tarefas de cálculo pendentes. Esses IDs não são armazenados ou enviados a destinos externos.
  • Cookies de sessão do SPX: Utilizados apenas para autenticar as requisições à API do próprio SPX Shopee (endpoint de cancelamento), dentro da aba já autenticada do usuário. A extensão não lê, armazena ou transmite esses cookies para nenhum destino externo.

4. Permissões Utilizadas

  • activeTab: Para interagir com a aba ativa do SPX quando o usuário clica na extensão.
  • scripting: Para executar scripts na página do SPX que leem dados da tabela de CTs e fazem requisições à API de cancelamento.

5. Comunicações de Rede

A extensão realiza requisições HTTP exclusivamente para o domínio spx.shopee.com.br, utilizando a sessão já autenticada do usuário. Especificamente, utiliza o endpoint /api/spx/lmroute/adminapi/calculation_task/cancel para cancelar CTs. Nenhuma requisição é feita para domínios de terceiros.

6. Armazenamento

A extensão não utiliza armazenamento persistente (localStorage, chrome.storage, IndexedDB ou similares). Todos os dados são mantidos apenas em memória durante o uso e descartados ao fechar o popup.

7. Alterações nesta Política

Eventuais alterações serão publicadas nesta página com a data de atualização revisada.

8. Contato

Em caso de dúvidas sobre esta política de privacidade, entre em contato pelo e-mail do desenvolvedor disponível na página da extensão na Chrome Web Store.
